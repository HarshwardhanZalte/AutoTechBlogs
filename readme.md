# AutoTechBlogs: AI-Powered Tech Blog Generator

This is a complete full-stack Django application that automatically fetches trending technology news from the GNews.io API, uses the Gemini AI to write unique blog posts, and serves them on a modern, dark-mode-ready website.

### Features
- Backend: Django & Django Rest Framework
- AI Integration: Google's Gemini API for blog content generation.
- News Source: GNews.io API for trending tech articles.
- Frontend: HTML5, CSS3, and modern JavaScript (no frameworks).

### Core Functionality:
- Automated blog generation.
- Home page with paginated blog cards.
- Full blog detail page.
- Dynamic, asynchronous comment system.

### Design:
- Modern, tech-inspired "neon" aesthetic.
- Dark Mode & Light Mode toggle.
- Responsive design for mobile and desktop.
